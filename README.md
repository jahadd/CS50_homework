# CS50 Homework

Repositorio con las tareas y prácticas del curso [CS50's Introduction to Computer Science](https://cs50.harvard.edu/x/) de Harvard.

## Contenido

### Problem Sets (C)
- **hello** — Primer programa en C
- **credit** — Validación de números de tarjeta (Luhn)
- **readability** — Índice de legibilidad (Coleman-Liau)
- **plurality** — Sistema de votación por pluralidad
- **runoff** — Votación por elección instantánea (runoff)
- **substitution** — Cifrado por sustitución
- **recover** — Recuperación de imágenes JPEG desde memoria
- **speller** — Corrector ortográfico con tablas hash y tries
- **filter-less** — Filtros de imagen (blur, bordes, etc.)
- **inheritance** — Herencia de tipo sanguíneo
- **volume** — Modificación de volumen de archivos de audio WAV
- **scrabble** — Puntuación de palabras (estilo Scrabble)

### Problem Sets (Python)
- **dna** — Identificación de personas por secuencias de ADN
- **sentimental-hello** — Hello World en Python
- **sentimental-mario-more** — Pirámide de Mario (más)
- **sentimental-cash** — Cambio mínimo (monedas)
- **sentimental-readability** — Legibilidad en Python

### Labs
- **sort** — Análisis de algoritmos de ordenamiento
- **friends** — SQL (consultas con bases de datos)

### Prácticas y otros
- **practicas** — Ejercicios de práctica en C
- **problems** — Problemas adicionales (p. ej. Mario)

## Uso

Cada carpeta corresponde a un problema o lab. Dentro suele haber:
- Código fuente (`.c`, `.py`)
- En algunos casos: `Makefile`, datos de prueba o archivos de apoyo

Para compilar y ejecutar, entra en la carpeta del problema y sigue las instrucciones del enunciado de CS50.
